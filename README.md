## Env Variables

Make Sure to Create a .env.local file in main directory and add appropriate variables in order to use the app.

**Essential Variables**
GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET,
JWT
MONGODB_URI
MONGODB_DB
